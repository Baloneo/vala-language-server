Properties:

  • public SymbolAccessibility access { get; set; }
    Specifies the accessibility of this symbol. Public accessibility doesn't limit access. Default accessibility limits access to this program or library. Private accessibility limits access to instances of the contained type.
  • public bool active { get; set; }
    Specifies whether this symbol is active.
  • public bool anonymous { get; set; }
    Specifies whether this symbol is anonymous and has no public defintion.
  • public Comment? comment { get; set; }
  • public bool external { get; set; }

  • public bool external_package { get; }

  • public bool from_commandline { get; }
    Specifies whether the implementation came from the commandline.
