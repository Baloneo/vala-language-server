GirParser

Object Hierarchy:

Description:

public class GirParser : CodeVisitor

Code visitor parsing all GIR source files.

Pipeline: 1) Parse metadata 2) Parse GIR with metadata, track unresolved GIR symbols, create Vala symbols 3) Reconciliate the tree by mapping tracked symbols 4) Process the tree
